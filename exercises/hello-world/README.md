# Hello World Exercise

A simple practice project to test your new setup.

## Goals

1. Practice tmux workflow
2. Test Claude Code
3. Build a simple Node.js server

## Instructions

Ask Claude Code to help you:

```
> create a simple hello world express server

> add a /health endpoint

> run the server and test it
```

## Expected Layout

```
┌─────────────────────────────────────┐
│           Claude Code               │
├──────────────────┬──────────────────┤
│   Code/Editor    │   Server logs    │
└──────────────────┴──────────────────┘
```

## Commands to Practice

1. `Ctrl+b |` - Split for server
2. `Ctrl+l` - Move to right pane
3. `Ctrl+h` - Move back to Claude
4. `Ctrl+b m` - Zoom to focus

Have fun!
